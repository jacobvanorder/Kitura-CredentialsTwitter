CredentialsTwitter

public class CredentialsTwitter: CredentialsPluginProtocol

CredentialsTwitter is a plugin for the Credentials framework that authenticate using Twitter. This plugin uses Twitter’s “Sign in with Twitter” use case. Roughly, there are three steps to this process: obtaining a request token, redirecting the user, and converting the request token to an access token. These three steps are marked down below. The goal of this process is create a UserProfile to give back to Credentials.
